# دمج التريليون 🔥🔴
لعبة مفتوحة المصدر مبنية باستخدام **HTML / CSS / JavaScript**، تهدف إلى تقديم تجربة ممتعة مع مستويات متدرجة الصعوبة، نظام عملات، وأسلحة قابلة للتطوير.

## 🎮 طريقة التشغيل
- افتح الملف `index.html` في أي متصفح حديث.
- استخدم لوحة المفاتيح:
  - W: التحرك للأمام
  - S: التراجع للخلف
  - A: التحرك يسارًا
  - D: التحرك يمينًا
- التصويب يتم عبر تحريك الماوس.
- عند الضغط بالماوس يتم إطلاق النار مع مؤثرات صوتية وبصرية.

## 🕹️ مميزات اللعبة
- 20 مستوى مع زيادة الصعوبة كل 10 مستويات.
- زيادة عدد البوتات وسرعتها وضرر أسلحتها تدريجيًا.
- واجهة HUD تعرض:
  - شريط الصحة (HP) لك ولزملائك والأعداء.
  - عدد العملات المكتسبة.
- مؤثرات صوتية ومرئية لإطلاق النار 🔥🔴.
- نظام دمج وتطوير الأسلحة باستخدام العملات.

## 💰 نظام العملات

- يمكن استخدام العملات في دمج الأسلحة أو شراء عناصر جديدة.
