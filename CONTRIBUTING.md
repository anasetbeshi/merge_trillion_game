# إرشادات المساهمة في لعبة دمج التريليون 🔥🔴

شكرًا لرغبتك في المساهمة! قبل البدء، يرجى قراءة هذه القواعد:

## 🟢 مسموح
- إعادة استخدام الكود وتطويره مع ذكر المصدر.
- إضافة ميزات جديدة (مستويات، أسلحة، مؤثرات).
- تحسين الأداء أو إصلاح الأخطاء.
- إرسال تعديلات عبر Pull Requests.
- كتابة كود نظيف ومنظم مع تعليقات واضحة.

## 🔴 ممنوع
- نشر بيانات شخصية أو حساسة داخل الكود.
- استخدام صور أو أصوات محمية بحقوق ملكية بدون إذن.
- إضافة أي كود ضار أو محتوى مخالف.
- تغيير الرخصة أو شروط المشروع بدون موافقة.
- إساءة استخدام المشروع لأغراض غير قانونية.

## 🛠️ خطوات المساهمة
1. انسخ المستودع (Fork).
2. أنشئ فرع جديد لتعديلاتك:  
   ```bash
   git checkout -b feature/new-feature
